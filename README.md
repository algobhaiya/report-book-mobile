# Report Book Mobile v1.0.0

## 🎉 Initial Release

Report Book Mobile is a productivity tracking application that helps users record and monitor their daily activities and achieve monthly goals.

### ✨ Features

* Daily productivity entry
* Custom field templates
* Monthly target management
* Monthly summary and progress tracking
* Offline-first data storage using SQLite
* Multiple user profiles on a single device
* Modern Material Design interface
* Fast and lightweight performance

### 📱 Requirements

* Android 10 (API 29) or later

### 📦 Installation

1. Download the APK from the Released APK section below.
2. Enable **Install unknown apps** on your Android device if prompted.
3. Install the APK.
4. Open Report Book Mobile and create your first profile.
